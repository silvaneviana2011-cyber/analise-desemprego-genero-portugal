# 📊 Análise do Desemprego por Género em Portugal

Este projeto analisa a evolução do desemprego entre homens e mulheres em Portugal entre 2014 e 2024.

## 🎯 Objetivo
Comparar a taxa de desemprego por género e identificar diferenças ao longo do tempo.

## 📂 Dados
- Fonte: PORDATA
- Dados tratados com Python (pandas)

## 📊 Dashboard
O dashboard foi desenvolvido no Power BI e inclui:
- Evolução do desemprego por género
- Diferença percentual (Gap)
- Indicadores chave (KPIs)

## 💡 Insight
Apesar da redução global do desemprego, a taxa feminina manteve-se consistentemente superior à masculina.
